# laundry-elevator

This repo contains the 3D printed components of a little elevator I built in the laundry shoot of my house.

In addition to the 3D printed components, I used various other components

 * Motor - [BRINGSMART 12V 12rpm DC Worm Gear Motor](https://www.amazon.com/dp/B07F8Y36PD?ref=ppx_yo2ov_dt_b_fed_asin_title)
 * String - [2 mm x 330 Feet Satin Nylon Cord](https://www.amazon.com/dp/B0CS2JT1PJ?ref=ppx_yo2ov_dt_b_fed_asin_title&th=1)
 * Alligator clips - [EBOOT 20 Pcs Alligator Clips Electrical](https://www.amazon.com/dp/B0CSYP99VY?ref=ppx_yo2ov_dt_b_fed_asin_title&th=1)
 * Switch - [Reverse Polarity Toggle Switch 12V 30A 6 Pin 3 Way ON/Off/ON DPDT Latching Switch](https://www.amazon.com/dp/B0D1279RYM?ref=ppx_yo2ov_dt_b_fed_asin_title&th=1)
 * Power supply - [ANTARCTICA GEAR Portable Battery Power Bank](https://www.amazon.com/dp/B0FMRNH4XT?ref=ppx_yo2ov_dt_b_fed_asin_title)
 * Power cable - [DC Power Pigtail Cable](https://www.amazon.com/dp/B07VW5VRX7?ref=ppx_yo2ov_dt_b_fed_asin_title&th=1)
 * Wall screws - [KURUI Drywall Anchors and Screws](https://www.amazon.com/dp/B08HDFNDRP?ref=ppx_yo2ov_dt_b_fed_asin_title&th=1)
